# 大宗商品贸易智能审查 · 展示站

一个对外能力展示页（中英双语）。单文件静态站点，纯前端，无任何依赖或外部请求。

- **本地预览**：直接用浏览器打开 `index.html`。
- **在线发布**：把本仓库交给任意静态托管即可（GitHub Pages / Gitee Pages / Netlify / Cloudflare Pages），站点根目录的 `index.html` 就是首页。
- **语言**：页面右上角一键切换中文 / English。
- **主题**：随访问者系统的浅色 / 深色自动适配。
